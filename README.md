# Show Me The Lyrics

- Description: A simple browser app where you can search for the lyrics of a song.

# User Stories
As a user, I would like to search the name of a singer/artist and song title in return, I would see the lyrics.

# Technologies Used
- HTML5
- CSS3
- Javascript
- Bootstrap
